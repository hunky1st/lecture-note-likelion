# 3교시 clamp, calc

- orientation 화면의 방향에 따라 미디어의 유형을 결정하는 기능
- device-aspect-ratio 비율별로 밀도를 다르게 지정하고 싶을 때 사용
- 미디어 쿼리의 조건식
  - not/only 조건이 여러개일때 미디어쿼리의 조건을 제한하거나 무시하는 설정
  - 콤마(,)로 구분하면 or라는 의미임
- @media조건은 다른 조건 이를테면 @support와 함께 중첩하여 사용할 수 있음
- tailwindcss에서 중단점(minimum-widt)별 수치를 확인
- clamp속성(font-size와 연동)진행 - 예제를 예로들면 어떤 값의 범위에서만 글씨크기를 변하게 하고 싶다는 의미가 됨
- width calc속성
- width: calc(500px + var(--spacing) \* 2)같은 식으로 커스텀설정과 합쳐서 쓸 수 있고, 사칙연산을 자유롭게 할 수 있음.
